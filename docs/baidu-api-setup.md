## 1. 注册百度账号

访问 [百度翻译开放平台](https://fanyi-api.baidu.com/)，点击右上角 **「登录/注册」**。

- 已有百度账号（百度网盘、百度贴吧等）可直接登录
- 没有则用手机号或邮箱注册一个
- **必须完成实名认证**（个人认证即可，免费）

---

## 2. 开通翻译 API 服务

登录后进入控制台，点击 **「开通服务」**。

百度翻译提供多种服务，**必须选择正确的那个**：

```
❌ 通用文本翻译 API     — 不能用（机器翻译，效果差）
❌ 垂直领域翻译 API     — 不能用
✅ 大模型文本翻译 API   — ← 选这个！
```

> **关键点：** 服务名称是 **「大模型文本翻译」**，不是「通用翻译」。只有大模型翻译支持 `model_type: "llm"`，翻译质量才能满足游戏需求。

点击「大模型文本翻译 API」→「立即开通」→ 选择 **「个人开发者」** → 填写表单 → 提交。

---

## 3. 获取 APP ID 和密钥

在控制台左侧菜单点击 **「开发者信息」** 或 **「管理控制台」**：

| 字段 | config.yaml 对应 | 说明 |
|------|-----------------|------|
| **APP ID**（或 APPID） | `baidu.app_id` | 应用唯一标识，一串数字 |
| **密钥**（或 Secret Key） | `baidu.secret_key` | 用于签名验证 |

> ⚠️ **密钥只会在创建时显示一次**，请立即保存。如果丢失，需要重置密钥。

---

## 4. 填入配置文件

打开 `config.yaml`，找到 `baidu` 部分，填入你的凭据：

```yaml
baidu:
  app_id: "你的APP_ID"       # ← 替换这里
  secret_key: "你的密钥"      # ← 替换这里
  model_type: "llm"          # 保持不变
  timeout: 15                # 保持不变
```

---

## 5. 验证配置

双击 `run.bat` 启动程序。成功时日志显示：

```
[INFO] batmud: Baidu LLM API OK. Test: hello world -> 你好世界
```

常见错误对照：

| 错误信息 | 原因 | 解决 |
|----------|------|------|
| `翻译API错误 [54001]` | 密钥不正确 | 核对 `secret_key` |
| `翻译API错误 [52003]` | APP ID 不对或未开通大模型翻译 | 检查 `app_id`，确认开通的是「大模型文本翻译」 |
| `翻译API错误 [54004]` | 免费额度用完 | 等下月重置，或充值 |
| `翻译API错误 [54003]` | QPS 超限 | 正常游戏不会触发 |
| `翻译API错误 [58004]` | 开通的是通用翻译而非大模型翻译 | 重新开通「大模型文本翻译 API」 |

## 7. 常见问题

### Q: 为什么必须用「大模型翻译」？
通用翻译是传统机器翻译，游戏文本效果差。大模型翻译能理解语境，质量接近人工。

### Q: 可以多人共用一个 API Key 吗？
不推荐。多人在线会叠加 QPS，可能触发频率限制。建议每人注册自己的账号，免费。

### Q: API 密钥泄露了怎么办？
登录百度控制台 → 开发者信息 → 重置密钥，更新 `config.yaml` 即可。
